## 🚀 À propos

Portfolio interactif présentant mes compétences, projets et expériences en développement web. Conçu avec une interface terminal-style unique et deux thèmes élégants.

**🔗 Site en ligne :** [moussandou.github.io/Portfolio](https://moussandou.github.io/Portfolio/)

---

## ✨ Fonctionnalités

- 🎨 **Dual Theme** - Mode Clair (Deep Ocean) et Mode Sombre (Midnight Blue)
- 📱 **Responsive Design** - Optimisé pour mobile, tablette et desktop
- 🖥️ **Interface Terminal** - Design inspiré des terminaux Unix
- 🎯 **Modules Draggables** - Fenêtres de terminal déplaçables
- 💬 **Guestbook Firebase** - Livre d'or en temps réel avec modération
- ⚡ **Performance Optimisée** - Lazy loading, images optimisées
- ♿ **Accessible** - Alt texts, ARIA labels, navigation clavier
- 🔍 **SEO Optimisé** - Meta tags, Open Graph, sitemap

---

## 🛠️ Stack Technique

### Frontend
- **React 18** - Bibliothèque UI
- **TypeScript** - Typage statique
- **Vite** - Build tool ultra-rapide
- **TailwindCSS** - Framework CSS utility-first

### Backend & Services
- **Firebase Firestore** - Base de données temps réel pour le guestbook
- **Firebase Hosting** - Hébergement sécurisé

### UI Components
- **React Slick** - Carrousels d'images pour les projets
- **Custom Components** - Composants sur mesure (AsciiArt, Terminal, etc.)

### Outils
- **ESLint** - Linter JavaScript/TypeScript
- **PostCSS** - Transformations CSS
- **gh-pages** - Déploiement GitHub Pages
- **GitHub Actions** - CI/CD

## 🎨 Thèmes

### Mode Clair - Deep Ocean
- Design professionnel avec palette vert océan
- Fond dégradé turquoise clair (`#E8F8F5` → `#D5F4EC` → `#C8F0E9`)
- Modules avec fond blanc pur
- Couleurs principales : `#0E6655` (primary), `#117A65` (accent)

### Mode Sombre - Midnight Blue
- Thème sombre moderne et élégant
- Fond bleu nuit profond (`#0A0E1A` → `#141829` → `#1A1F2E`)
- Modules avec fond bleu foncé (`#0F1729`)
- Couleurs principales : `#5DADE2` (primary), `#85C1E9` (accent)
- Effets glow bleus subtils
