## 🚀 À propos

Portfolio interactif présentant mes compétences, projets et expériences en développement web. Conçu avec une interface terminal-style unique et deux thèmes (Mode Clair / Hack Mode).

**🔗 Site en ligne :** [moussandou.github.io/Portfolio](https://moussandou.github.io/Portfolio/)

---

## ✨ Fonctionnalités

- 🎨 **Dual Theme** - Mode Clair élégant et Hack Mode cyberpunk
- 📱 **Responsive Design** - Optimisé pour mobile, tablette et desktop
- 🖥️ **Interface Terminal** - Design inspiré des terminaux Unix
- 🎯 **Modules Draggables** - Fenêtres de terminal déplaçables
- ⚡ **Performance Optimisée** - Lazy loading, images optimisées
- ♿ **Accessible** - Alt texts, ARIA labels, navigation clavier
- 🔍 **SEO Optimisé** - Meta tags, Open Graph, sitemap

---

## 🛠️ Stack Technique

### Frontend
- **React 18** - Bibliothèque UI
- **TypeScript** - Typage statique
- **Vite** - Build tool ultra-rapide
- **TailwindCSS** - Framework CSS utility-first

### UI Components
- **Radix UI** - Composants accessibles headless
- **Lucide React** - Icônes modernes
- **React Slick** - Carrousels d'images
- **Recharts** - Graphiques React

### Outils & DevOps
- **ESLint** - Linter JavaScript/TypeScript
- **PostCSS** - Transformations CSS
- **gh-pages** - Déploiement automatique
- **GitHub Actions** - CI/CD

## 📦 Installation

### Prérequis
- Node.js >= 18.x
- npm >= 9.x

### Clone et Installation

\`\`\`bash
# Cloner le repository
git clone https://github.com/Moussandou/Portfolio.git
cd Portfolio

# Installer les dépendances
npm install
\`\`\`

---

## 🚀 Utilisation

### Développement Local

\`\`\`bash
# Lancer le serveur de développement
npm run dev

# Ouvrir http://localhost:5173
\`\`\`

### Build de Production

\`\`\`bash
# Créer le build optimisé
npm run build

# Prévisualiser le build
npm run preview
\`\`\`

### Déploiement

\`\`\`bash
# Déployer sur GitHub Pages
npm run deploy
\`\`\`

### Linting

\`\`\`bash
# Vérifier le code
npm run lint
\`\`\`
